# Job Notification Tracker - Route Skeleton

## Overview

This is the route skeleton for the **Job Notification Tracker** application, built within the KodNest Premium Build System design framework.

## Routes Implemented

All routes are functional with placeholder pages:

- **`/`** - Dashboard (default route)
- **`/dashboard`** - Dashboard
- **`/saved`** - Saved notifications
- **`/digest`** - Digest view
- **`/settings`** - Settings page
- **`/proof`** - Proof section

## Navigation

### Desktop Navigation
Horizontal navigation bar with links to all routes. Active route is highlighted with a deep red underline (#8B0000).

### Mobile Navigation
Responsive hamburger menu that collapses navigation on screens ≤768px. Menu slides open/closed with smooth transitions.

✅ **Off-white background**: #F7F6F3

## File Structure

```
app.html        - Main application HTML with navigation
app.css         - Application-specific styles
router.js       - Client-side routing logic
```

## How to Use

1. Open `app.html` in a web browser
2. Navigate between routes using the top navigation
3. On mobile, use the hamburger menu to access routes
