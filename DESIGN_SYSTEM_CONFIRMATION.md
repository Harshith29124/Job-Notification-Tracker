## Design Philosophy Implementation

### ✅ Calm, Intentional, Coherent, Confident
- No flashy effects or loud colors

## Color System ✅

**Maximum 4 colors used:**
1. **Background**: `#F7F6F3` (off-white)
2. **Primary Text**: `#111111` (near black)
3. **Accent**: `#8B0000` (deep red)
4. **Supporting**: Muted green (`#2D5016`) and amber (`#8B6914`)

All colors are calm, muted, and professional.

---

## Typography ✅

### Fonts
- **Headings**: Crimson Pro (Serif) - Large, confident, generous spacing
- **Body**: Inter (Sans-serif) - 16–18px, line-height 1.6–1.8

### Text Width
- Maximum 720px for comfortable reading
- No decorative fonts
- No random sizes

---

## Spacing System ✅

**Consistent 8px-based scale:**
- 8px (xs)
- 16px (sm)
- 24px (md)
- 40px (lg)
- 64px (xl)

**No random spacing values used.** Whitespace is intentional.

## Global Layout Structure ✅

Every page follows the exact structure:

```
[Top Bar]
  ├─ Left: Project name
  ├─ Center: Progress indicator (Step X / Y)
  └─ Right: Status badge

[Context Header]
  ├─ Large serif headline
  └─ 1-line subtext

[Primary Workspace (70%) | Secondary Panel (30%)]
  ├─ Primary: Main interaction area
  └─ Secondary: Step explanation, prompt box, action buttons

[Proof Footer]
  └─ Checklist: □ UI Built □ Logic Working □ Test Passed □ Deployed
```

---

## Component Rules ✅

### Buttons
- Primary: Solid deep red
- Secondary: Outlined
- Same hover effect (150ms ease-in-out)
- Same border radius (4px)

### Inputs
- Clean borders
- Clear focus state (border changes to primary text color)
- No heavy shadows

### Cards
- Subtle border (1px solid)
- No drop shadows
- Balanced padding (40px main, 24px nested)

---

## Interaction Rules ✅

### Transitions
- Speed: 150ms
- Timing: ease-in-out
- No bounce, no parallax

### States
- Hover: Subtle color/border changes
- Focus: Clear border indication
- Disabled: 40% opacity

---

## Error & Empty States ✅

### Errors
- Explain what went wrong
- Provide how to fix
- Never blame user
- Muted red background

### Empty States
- Provide next action
- Clear CTA button
- Never feel abandoned
